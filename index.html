DIA Samples

A collection of interactivty-oriented samples to accompany Kattegat. The -playground samples are meant as little experimental platforms, mostly to help you to visualise data input, or to manually control something. Unlike the rest of the samples, they are not meant to be extended.

As you play with the samples keep your Javascript console open, read the README.md file and browse the source code.

Note These links - and most of the demos themselves - expect that you are accessing them via your Kattegat server, eg, from http://localhost:3000/dia-samples/.

Basics

square

Basic text manipulation, events, creating HTML elements, assigning CSS properties and transitioning CSS values.

square-store

Storing and fetch data in the browser or on the server

square-decouple

Controlling the square creation from another page - or potentially another server. Demonstrates Kattegat's realtime communication capabilities

square-direct

Demonstrates how to make a more direct interaction, using more keyboard events and fancier CSS.

Touch + Pointers

pointer

Setting CSS properties on the basis of pointer up and down events, and also how to track whether the pointer was held down

pointer-drag

Visualises the pointer position, and shows how to move an element around to match the pointer

pointer-playground

Play with pointer events and see what data the event holds

pointer-world

Broadcast pointermove events to other windows/devices

touch-swipe

Change colour based on the velocity of swipe gesture

touch-solar

Create and manipulate HTML elements based on touch events and gestures

touch-playground

Play with touch gesture events and see what data the event holds

touch-mix

Mix colour and transform an element from different devices

Audio

audio-play

Play a sample and sequence synchronised sample playback

audio-filter

Loop a sample and filter the sound based on pointer movement

audio-pan

Start one or more loops and change how they are panned in the soundscape

audio-synth

Manipulate frequency and volume with a pad

audio-oscillator

Control an audio oscillator according to the distance of one object to another

audio-input

Use sound levels from the microphone as an input

speech-input

Execute commands with your speech

speech-outout

Use text-to-speech

Geolocation

geo-playground

See your current coordinates and find out coordinates of places on a map

geo-trip

Demonstrates basic calculation between coordinates. In this case, showing distance and average speed from a start point

geo-zones

Define spatial zones and detect whether the user is inside of them, and their distance away. Useful for location-based experiences.

geo-proximity

Track distance between multiple people/devices

Motion + Orientaiton

motion-playground

See different output over time. acceleration, accelerationIncludingGravity and rotationRate

motion-force

Control a square in contained space

motion-sound

Adjust a sound with motion

motion-multiple

Connect multiple devices and use Smoother to analyze motion data

orientation-playground

Measure the current orientation of a device

Computer Vision

cv-face

Track the movement of a face with a device's in-built camera
