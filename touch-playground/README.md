# Touch playground

This demo shows different kinds of [Hammerjs](http://eightmedia.github.io/hammer.js/) events. The full list is:
- touch
- release
- hold
- tap
- doubletap
- drag
- dragstart
- dragend
- dragleft
- dragright
- dragup
- dragdown
- swipe
- swipeleft
- swiperight
- swipeup
- swipedown
- transformstart
- transform
- transformend
- rotate
- rotateleft
- rotateright
- pinch
- pinchin
- pinchout
